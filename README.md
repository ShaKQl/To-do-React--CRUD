# TODO React + Vite CRUD program

This project provides a minimal setup to get React to-do app in Vite with HMR and some ESLint rules.

For icons was used the service of the:
-http://heroicons.com/

## React Compiler

The React Compiler is not enabled on this template because of its impact on dev & build performances. 

## Expanding the ESLint configuration

If the developing will be proceeded to a production application, the use of TypeScript with type-aware lint rules is 'must' in implemention.
